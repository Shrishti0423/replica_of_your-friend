HOW TO CONVERT YOUR CHAT TO JSON
=================================

The replica system accepts this JSON format:

[
  {"role": "user", "content": "your message"},
  {"role": "assistant", "content": "their response"}
]

METHOD 1: EXPORT FROM WHATSAPP (Android/iPhone)
================================================

1. Open WhatsApp chat
2. Go to Options (3 dots) > More > Export chat
3. Choose "Without media" 
4. Save the .txt file
5. Convert using the script below

METHOD 2: EXPORT FROM TELEGRAM
===============================

1. Open Telegram Desktop
2. Go to Settings > Advanced > Export data
3. Select chat/contacts to export
4. Download JSON file (already in right format!)
5. Use directly with /create command

METHOD 3: EXPORT FROM FACEBOOK MESSENGER
=========================================

1. Go to facebook.com/download
2. Click "Download Your Information"
3. Select "Messages"
4. Download HTML file
5. Convert using the script below

METHOD 4: COPY-PASTE TEXT (Easy)
=================================

If you just have text chat:

Person1: Hello
Person2: Hi there
Person1: How are you?
Person2: Doing great!

Save as .txt file, then use the conversion script below.

AUTOMATIC CONVERSION SCRIPT
============================

Create a file named convert_to_json.py with this content:

---

import json
import sys

def convert_whatsapp_to_json(text_file, person_name):
    """Convert WhatsApp export to JSON"""
    messages = []
    
    with open(text_file, 'r', encoding='utf-8') as f:
        lines = f.readlines()
    
    for line in lines:
        line = line.strip()
        if not line or ' - ' not in line:
            continue
        
        # Parse: [timestamp] Name: message
        try:
            # Remove timestamp [HH:MM, DD/MM/YYYY] 
            if line.startswith('['):
                line = line.split('] ', 1)[1]
            
            if ': ' not in line:
                continue
            
            name, message = line.split(': ', 1)
            
            # Determine role
            if name == person_name:
                role = 'user'
            else:
                role = 'assistant'
            
            messages.append({
                'role': role,
                'content': message
            })
        except:
            continue
    
    return messages

def convert_generic_text_to_json(text_file, user_name='User', assistant_name='Assistant'):
    """Convert generic text format to JSON"""
    messages = []
    
    with open(text_file, 'r', encoding='utf-8') as f:
        lines = f.readlines()
    
    for line in lines:
        line = line.strip()
        if not line or ': ' not in line:
            continue
        
        try:
            name, message = line.split(': ', 1)
            
            if name == user_name:
                role = 'user'
            elif name == assistant_name:
                role = 'assistant'
            else:
                role = 'user'  # default
            
            messages.append({
                'role': role,
                'content': message
            })
        except:
            continue
    
    return messages

# Usage
if __name__ == "__main__":
    # Example 1: Convert WhatsApp chat
    messages = convert_whatsapp_to_json('chat.txt', 'Your Name')
    
    with open('chat.json', 'w', encoding='utf-8') as f:
        json.dump(messages, f, indent=2, ensure_ascii=False)
    
    print(f"Converted {len(messages)} messages to chat.json")

---

STEP-BY-STEP GUIDE
==================

Step 1: Get Your Chat File
   - Export from WhatsApp, Telegram, or messenger
   - Save as .txt file
   - Example: chat.txt

Step 2: Create conversion script
   - Save the code above as convert_to_json.py
   - Put it in your chatbot folder

Step 3: Edit the script
   - Change 'Your Name' to YOUR ACTUAL NAME
   - Example: convert_whatsapp_to_json('chat.txt', 'John')

Step 4: Run the conversion
   python convert_to_json.py

Step 5: Check the output
   - Should create chat.json
   - Open it to verify format

Step 6: Use with replica system
   python replica_main.py
   /create
   Name: Person Name
   File: chat.json
   Format: json

2. Type:

  {"role": "user", "content": "Message 1"},
  {"role": "assistant", "content": "Response 1"},

]

3. Save as chat.json
4. Done!

VERIFY YOUR JSON
================

After creating JSON file:

1. Open in Notepad
2. Check format:
   - Starts with [
   - Ends with ]
   - Each entry has "role" and "content"
   - Proper commas between entries

Example of CORRECT format:
[
  {"role": "user", "content": "hello"},
  {"role": "assistant", "content": "hi there"}
]

Example of WRONG format:
[
  {"role": "user", "content": "hello"}
  {"role": "assistant", "content": "hi there"}  <- Missing comma!
]

COMMON ISSUES
=============

Issue: "Invalid JSON" error
Fix: Check for missing commas or quotes

Issue: Role is wrong
Fix: Make sure "user" and "assistant" are correct
     user = person being replicated
     assistant = their responses

Issue: Special characters not working
Fix: Make sure file is saved as UTF-8
     In Notepad: Save As > Encoding > UTF-8

Issue: Can't find file
Fix: Make sure file is in chatbot folder
     Or use full path: c:\path\to\chat.json

TIPS
====

1. Remove timestamps - they're not needed
2. Remove system messages (like "User changed name")
3. Keep messages in chronological order
4. Make sure roles alternate (user, assistant, user, etc.)
5. Use at least 10-20 messages for better analysis
