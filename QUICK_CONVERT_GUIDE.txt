QUICK GUIDE - Convert Chat to JSON
===================================

FASTEST METHOD (Copy/Paste):
============================

1. Open your chat (WhatsApp, Messenger, etc)
2. Copy the messages (without timestamps if possible)
3. Paste into Notepad
4. Save as: chat.txt
5. Format should be:
   John: Hello
   Jane: Hi there!
   John: How are you?
   Jane: I'm good!

6. Run the converter:
   python convert_to_json.py

7. Answer questions:
   Which format? Type: 2 (plain text)
   File path? Type: chat.txt
   Person's name? Type: John

8. Check result: chat.json created!

9. Use with replica:
   python replica_main.py
   /create
   Name: John
   File: chat.json
   Format: json

WHATSAPP EXPORT METHOD:
=======================

1. Open WhatsApp
2. Select conversation
3. Menu (three dots) > More > Export Chat
4. Choose "Without media"
5. Save file (chat.txt)

6. Run converter:
   python convert_to_json.py

7. Answer:
   Format: 1 (WhatsApp)
   File: chat.txt
   Name: Your Name (as it shows in chat)

8. Done! Use with replica system

JSON FORMAT REFERENCE:
======================

Correct format:
[
  {"role": "user", "content": "Hello"},
  {"role": "assistant", "content": "Hi there"},
  {"role": "user", "content": "How are you?"},
  {"role": "assistant", "content": "I'm good!"}
]

Notes:
- "role": "user" = messages FROM the person
- "role": "assistant" = messages TO the person
- Messages alternate between user and assistant
- Use double quotes, not single quotes

USING THE CONVERTER SCRIPT:
===========================

Interactive mode (easiest):
  python convert_to_json.py
  Follow the prompts

Command line mode:
  python convert_to_json.py chat.txt whatsapp "John"
  python convert_to_json.py chat.txt text "Jane"

TIPS FOR BEST RESULTS:
======================

1. Use at least 10-20 messages
   More messages = better personality analysis

2. Remove system messages
   (like "User joined" or "Group created")

3. Remove media descriptions
   (keep only text messages)

4. Keep chronological order
   (oldest first, newest last)

5. Use real messages
   The more authentic, the better the replica

6. Check the JSON file
   Make sure it looks correct before using

TROUBLESHOOTING:
================

Problem: "File not found"
Solution: Make sure file is in same folder
         Or use full path: c:\path\to\chat.txt

Problem: "Invalid role"
Solution: Check person's name matches exactly
         Include correct capitalization

Problem: "No messages converted"
Solution: Check file format
         Should be: Name: message
         Make sure there are colons

Problem: "JSON error when running replica"
Solution: Open chat.json in Notepad
         Check for proper formatting
         Make sure commas are correct

MANUAL JSON CREATION:
====================

If converter doesn't work:

1. Open Notepad
2. Create file with this format:

[
  {"role": "user", "content": "message 1"},
  {"role": "assistant", "content": "response 1"},
  {"role": "user", "content": "message 2"},
  {"role": "assistant", "content": "response 2"}
]

3. Save as: chat.json
4. Use with replica system

That's it!
